# Quick Install

```
$ python3 -m venv venv38

$ . venv/bin/activate

$ python --version
Python 2.7.18

$ python3 --version
Python 3.8.2

$ pip3 install --upgrade pip

$ pip3 install Flask

$ pip3 install six

$ mkdir .creds
$ vi .creds/creds.json

$ export FLASK_APP=lunchandlearn.py

$ flask run
```
